Лабораторная работа №3
Тема: «Создание и управление потоками. Синхронизация потоков»
Программа для многопоточной обработки массива с использованием потоков стандартной библиотеки C++. Процесс состоит из потока main и нескольких экземпляров потока marker, которые совместно работают с общим массивом целых чисел.
Поток main выполняет:
Захват памяти под массив целых чисел, размерность вводится с консоли
Инициализацию элементов массива нулями
Запрос количества потоков marker для запуска
Запуск заданного количества экземпляров потока marker с передачей порядкового номера
Сигнал на начало работы всех потоков marker
Циклическое выполнение:
Ожидание сигналов о невозможности продолжения от всех потоков marker
Вывод содержимого массива на консоль
Запрос порядкового номера потока marker для завершения
Сигнал завершения выбранному потоку
Ожидание завершения работы потока
Вывод обновленного массива на консоль
Сигнал продолжения работы оставшимся потокам
Завершение работы после окончания всех потоков marker
Поток marker выполняет:
Начало работы по сигналу от потока main
Инициализацию генератора случайных чисел с использованием своего порядкового номера
Циклическую работу:
Генерацию случайного числа и вычисление индекса в массиве
Если элемент равен нулю:
Задержку 5 мс
Запись своего номера в элемент
Задержку 5 мс
Продолжение цикла
Если элемент занят:
Вывод информации: номер потока, количество помеченных элементов, индекс проблемного элемента
Сигнал main о невозможности продолжения
Ожидание ответного сигнала
При сигнале завершения:
Заполнение нулями всех помеченных элементов
Завершение работы
При сигнале продолжения - возобновление цикла

Проект создан с использованием Windows API на С++ 17, сборка осуществляется с помощью системы CMake.
