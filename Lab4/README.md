Лабораторная работа №4
Тема: “Синхронизация процессов”

Программа для передачи сообщений между процессами через общий файл, которая включает в себя один процесс Receiver и несколько процессов Sender, которые посылают сообщения процессу Receiver. 

Процесс Receiver выполняет: 
Ввод с консоли имени бинарного файла и количество записей в бинарном файле, который будет использоваться для передачи сообщений. 
Создание бинарного файла для сообщений (до 20 символов).
Ввод с консоли количество процессов Sender.
Запуск заданного количества процессов Sender. Передача каждому процессу Sender через командную строку имя файла для сообщений.
Ожидание сигнала на готовность к работе от всех процессов Sender. Циклическое выполнение следующих действия по команде с консоли: читать сообщение из бинарного файла или завершить свою работу. 

Процесс Sender выполняет: 
Открытие файла для передачи сообщений. Имя файла получить из командной строки.
Отправление процессу Receiver сигнала на готовность к работе. 
Циклическое выполнение следующие действия по команде с консоли: отправить процессу Receiver сообщение или прекратить свою работу. Текст сообщения вводится с консоли (до 20 символов).

Проект создан с использованием Windows API на С++ 17, сборка осуществляется с помощью системы CMake.
